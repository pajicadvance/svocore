# SVO Core

This is a core mod used in my modpack [Simple Vanilla Overhaul (SVO)](https://modrinth.com/modpack/svo). It's a cut down version of [Affogato Core](https://modrinth.com/mod/affogato-core) with fewer patches and features.

Feel free to use in your own modpack. All features are disabled by default and are configured via [Fzzy Config](https://modrinth.com/mod/fzzy-config).
